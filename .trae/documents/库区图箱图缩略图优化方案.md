# 库区图/箱图缩略图优化方案（最终版）

## 一、范围限定

只优化**库区图和箱图**（有点击放大功能兜底），快麦 SKU 图（`picPath`）不做更改。

| 图片类型 | 缩略图优化 | 点击放大 | 理由 |
|:---------|:----------:|:--------:|:------|
| 库区图/箱图 | ✅ 后端生成200px缩略图 | ✅ 支持 | 有兜底，缩略图糊了可以点开看全尺寸 |
| SKU图(picPath) | ❌ 保持原样 | ❌ 无此功能 | 无放大入口，缩略图即最终显示，需保持清晰度 |

## 二、修改内容

### 修改 1（后端）：上传时自动生成缩略图

**文件**：`backend/app/routers/images.py` — 新增 `_generate_thumbnail()`

```python
from PIL import Image

def _generate_thumbnail(file_path: str) -> None:
    """生成200px宽度的缩略图，失败不影响主流程"""
    try:
        img = Image.open(file_path)
        img.thumbnail((200, 200), Image.LANCZOS)
        base, _ = os.path.splitext(file_path)
        thumb_path = f"{base}_thumb.jpg"
        img.convert("RGB").save(thumb_path, "JPEG", quality=50)
        logger.info(f"缩略图生成成功: {os.path.basename(thumb_path)}")
    except Exception as e:
        logger.warning(f"生成缩略图失败: {e}")
```

在 `upload_image()` 中写入全尺寸文件后调用：

```python
# 写入全尺寸文件...
with open(file_path, "wb") as f:
    f.write(content)

# 生成缩略图（失败不影响主流程）
_generate_thumbnail(file_path)

# 继续DB操作...
```

**缩略图文件**：`B08-24_area_a1b2c3d4.jpg` → `B08-24_area_a1b2c3d4_thumb.jpg`（同目录，~5-15KB）

### 修改 2（Android）：PickDetailViewModel 提供缩略图 URL

**文件**：`app/src/main/java/.../PickDetailViewModel.kt`

新增 data class 和辅助方法：

```kotlin
data class ImageUrls(
    val areaUrl: String?,
    val boxUrl: String?,
    val areaThumbUrl: String?,
    val boxThumbUrl: String?
)

/** 从完整URL构造缩略图URL（在扩展名前插入 _thumb） */
private fun buildThumbUrl(fullUrl: String?): String? {
    if (fullUrl == null) return null
    val dot = fullUrl.lastIndexOf('.')
    return if (dot > 0) "${fullUrl.substring(0, dot)}_thumb${fullUrl.substring(dot)}" else fullUrl
}
```

修改 `getImageUrls()` 返回 `ImageUrls` 而非 `Pair`：

```kotlin
suspend fun getImageUrls(skuOuterId: String): ImageUrls {
    return try {
        val areaImage = imageRepository.getImageBySkuAndType(skuOuterId, AppConstants.IMAGE_TYPE_AREA)
        val boxImage = imageRepository.getImageBySkuAndType(skuOuterId, AppConstants.IMAGE_TYPE_BOX)
        val serverUrl = prefs.getString(PrefsKeys.KEY_SERVER_URL, AppConstants.DEFAULT_SERVER_URL)
            ?.trim() ?: AppConstants.DEFAULT_SERVER_URL
        val areaUrl = areaImage?.let { url ->
            if (serverUrl.isNotEmpty()) "${serverUrl.trimEnd('/')}/${url.imageUrl}" else url.imageUrl
        }
        val boxUrl = boxImage?.let { url ->
            if (serverUrl.isNotEmpty()) "${serverUrl.trimEnd('/')}/${url.imageUrl}" else url.imageUrl
        }
        ImageUrls(
            areaUrl = areaUrl,
            boxUrl = boxUrl,
            areaThumbUrl = buildThumbUrl(areaUrl),
            boxThumbUrl = buildThumbUrl(boxUrl)
        )
    } catch (e: Exception) {
        Log.w("PickDetailViewModel", "获取图片URL失败: ${e.message}")
        ImageUrls(null, null, null, null)
    }
}
```

### 修改 4（Android）：PickDetailScreen 传递缩略图 URL

**文件**：`app/src/main/java/.../PickDetailScreen.kt`

```kotlin
var areaImageUrl by remember { mutableStateOf<String?>(null) }
var boxImageUrl by remember { mutableStateOf<String?>(null) }
var areaThumbUrl by remember { mutableStateOf<String?>(null) }  // 新增
var boxThumbUrl by remember { mutableStateOf<String?>(null) }   // 新增

LaunchedEffect(item.skuOuterId) {
    val urls = viewModel.getImageUrls(item.skuOuterId)
    areaImageUrl = urls.areaUrl
    boxImageUrl = urls.boxUrl
    areaThumbUrl = urls.areaThumbUrl
    boxThumbUrl = urls.boxThumbUrl
}

PickItemRow(
    areaImageUrl = areaImageUrl,
    boxImageUrl = boxImageUrl,
    areaThumbUrl = areaThumbUrl,
    boxThumbUrl = boxThumbUrl,
    onAreaImageClick = {
        if (areaImageUrl != null) {
            previewImageUrl = areaImageUrl  // 完整URL -> 大图预览
            previewImageLabel = "库区图 - ${item.skuOuterId}"
        }
    },
    onBoxImageClick = {
        if (boxImageUrl != null) {
            previewImageUrl = boxImageUrl  // 完整URL -> 大图预览
            previewImageLabel = "箱图 - ${item.skuOuterId}"
        }
    }
)
```

## 四、回归风险

| 风险 | 分析 | 等级 |
|:-----|:-----|:----:|
| 旧图片无缩略图 | 字符串替换的URL不存在→Coil 404→灰底显示 | 低（下次重传后自动生成） |
| 缩略图生成失败 | catch 异常，不影响上传和 DB 操作 | 无影响 |
| 旧版APK无thumbUrl参数 | PickItemRow 参数默认值 `null` → 不走 thumbUrl 分支 → 保持原行为 | 无影响 |
| SKU图清晰度 | **不做修改**，保持原 CDN 直连 | ✅ 无风险 |
| 点击放大未退化 | 回调仍传递 `areaImageUrl`（全尺寸完整URL） | ✅ 已验证 |
| `buildThumbUrl` 空值 | `fullUrl == null` 直接 `return null` | ✅ 安全 |
