### 时序验证

```
修改前（当前）:

LoginScreen.login()
  → POST /api/users/login → 200 OK → token
  → syncKuaimaiCredentials(token) [async]
      → GET /api/kuaimai/credentials → check_permission("settings")
        → 普通用户没有 settings → 403 Forbidden
      → catch → Log.w → 静默失败 → 本地凭证=空
  → Login屏 → HomeScreen → 用户操作所有依赖快麦API的功能都因凭证为空而失败

修改后:

LoginScreen.login()
  → POST /api/users/login → 200 OK → token
  → syncKuaimaiCredentials(token) [async]
      → GET /api/kuaimai/credentials → get_current_user
        → 已登录用户 → 200 OK → 返回凭证
      → appKey.isNotEmpty() → true → 保存到本地
  → Login屏 → HomeScreen → 快麦API功能正常
```
