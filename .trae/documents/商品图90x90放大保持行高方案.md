# 商品图 90×90 放大 + 行高保持方案

## 修改内容（2 处）

### 修改 1：图片尺寸 90×90dp

**PickItemRow.kt L87**
```
// 修改前
.size(width = 100.dp, height = 100.dp)
// 修改后
.size(width = 90.dp, height = 90.dp)
```

### 修改 2：垂直 padding

**PickItemRow.kt L80**
```
// 修改前
.padding(horizontal = 12.dp, vertical = 10.dp)
// 修改后
.padding(horizontal = 12.dp, vertical = 4.dp)
```

### 行高核算

| 元素 | 值 |
|:-----|:--:|
| 图片高 | 90dp |
| 垂直 padding | 4dp×2 = 8dp |
| **行高** | **90 + 8 = 98dp ✅** |

图片尺寸从 52×52 → 90×90（宽高各增加约 1.73 倍），其余所有布局不变（Spacer、字体、右列）。
