# Web后台底部增加版本号信息

## 一、探索结果

管理后台由 [`admin.py`](file:///d:/trea项目/快麦取货通/backend/app/routers/admin.py) 中的 `_build_admin_html()` 函数返回完整 HTML 字符串。

当前状态：
- 已有 `_load_version_info()` 函数读取 `APK_VERSION_FILE` 中的版本号
- 已有 `APK_VERSION_FILE` 常量导入
- 版本信息（`currentVersion`）已存在于 JSON 文件中，与 `BUILD_VERSION` 一致

---

## 二、修改方案

**仅 1 处修改**：在 `_build_admin_html()` 函数中，调用 `_load_version_info()` 获取版本号，在页面底部 `<footer>` 中展示。

### 修改：`admin.py`

**位置**：`_build_admin_html()` 函数中，在 `api_key_status` 赋值之后（L153），添加：

```python
version_info = _load_version_info()
build_version = version_info.get("currentVersion", "") if version_info else ""
```

**位置**：在 `</body>` 之前（L833），添加：

```html
<div class="footer">
  <span>快麦取货通 v{build_version}</span>
</div>
```

**在 CSS 中新增 footer 样式**（在 `.checkbox-group` 之后）：

```css
.footer {{ text-align:center; padding:20px; font-size:12px; color:#999; border-top:1px solid #e5e7eb; margin-top:24px; }}
```
