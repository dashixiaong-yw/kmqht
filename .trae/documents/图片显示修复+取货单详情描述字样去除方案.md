# 图片显示修复 + 取货单详情描述字样去除方案

## 一、问题分析

### 问题 1：App 端图片上传后不显示（后端可查看）

**根因**：URL 拼接时缺少 `/` 分隔符。

后端返回的 `imageUrl` 为 `"images/20260623/xxx.jpg"`（**无前导 `/`**），而 `serverUrl` 为 `"https://frp-off.com:64623"`（**无尾随 `/`**）。当前代码：

```kotlin
"${serverUrl.trimEnd('/')}${url.imageUrl}"
// 结果: "https://frp-off.com:64623images/20260623/xxx.jpg"
//                                     ↑ 缺少 / 分隔符
// 正确: "https://frp-off.com:64623/images/20260623/xxx.jpg"
```

**影响范围**：所有 3 个拼接点都缺少 `/`：

| 位置 | 文件 | 行号 |
|:-----|:-----|:----:|
| loadImages areaImageUrl | ProductViewModel.kt | L208 |
| loadImages boxImageUrl | ProductViewModel.kt | L211 |
| uploadImage 后拼接 | ProductViewModel.kt | L413 |
| getImageUrls areaUrl | PickDetailViewModel.kt | L476 |
| getImageUrls boxUrl | PickDetailViewModel.kt | L477 |

**注意**：SKU 主图（`picPath`）来自快麦 API 返回的完整 CDN URL（如 `https://img.alicdn.com/...`），不走 URL 拼接，因此正常显示。

### 问题 2：取货单详情中库区图与箱图的描述字样

PickItemRow.kt L150-L168（库区图）和 L187-L204（箱图）中，每个缩略图底部有一个 18dp 的半透明背景 Text("库区") / Text("箱图")。用户希望去掉这些文字标签。

## 二、修改内容

### 修改 1：修复 URL 拼接（ProductViewModel.kt + PickDetailViewModel.kt）

**文件 A**：`app/src/main/java/com/kuaimai/pda/ui/product/ProductViewModel.kt`

3 处修改，均将 `"${serverUrl.trimEnd('/')}${url.imageUrl}"` 改为 `"${serverUrl.trimEnd('/')}/${url.imageUrl}"`：

```kotlin
// L208
if (serverUrl.isNotEmpty()) "${serverUrl.trimEnd('/')}/${url.imageUrl}" else url.imageUrl

// L211
if (serverUrl.isNotEmpty()) "${serverUrl.trimEnd('/')}/${url.imageUrl}" else url.imageUrl

// L413
val fullUrl = if (serverUrl.isNotEmpty()) "${serverUrl.trimEnd('/')}/$imageUrl" else imageUrl
```

**文件 B**：`app/src/main/java/com/kuaimai/pda/ui/pickdetail/PickDetailViewModel.kt`

2 处修改：

```kotlin
// L476
val areaUrl = areaImage?.let { url -> if (serverUrl.isNotEmpty()) "${serverUrl.trimEnd('/')}/${url.imageUrl}" else url.imageUrl }

// L477
val boxUrl = boxImage?.let { url -> if (serverUrl.isNotEmpty()) "${serverUrl.trimEnd('/')}/${url.imageUrl}" else url.imageUrl }
```

### 修改 2：去掉取货单详情中图片描述字样（PickItemRow.kt）

**文件**：`app/src/main/java/com/kuaimai/pda/ui/components/PickItemRow.kt`

**库区图**（L134-L169）：去掉 L151 的占位 Text("库区")，去掉 L153-L168 的底部半透明标签 Box。

```kotlin
// 修改后
Box(
    modifier = Modifier
        .size(44.dp)
        .clip(RoundedCornerShape(6.dp))
        .background(SurfaceGray)
        .clickable { onAreaImageClick() },
    contentAlignment = Alignment.Center

    if (!areaImageUrl.isNullOrEmpty()) {
        AsyncImage(
            model = areaImageUrl,
            contentDescription = "库区图",
            modifier = Modifier.fillMaxSize(),
            contentScale = ContentScale.Crop
        )
    }
}
```

**箱图**（L171-L205）：去掉 L187 的占位 Text("箱图")，去掉 L189-L204 的底部半透明标签 Box。

```kotlin
// 修改后
```

## 三、回归风险分析

| 风险 | 分析 | 等级 |
|:-----|:-----|:----:|
| `imageUrl` 已含前导 `/` | 后端固定生成 `"images/..."` 无前导 `/`，新增 `/` 是安全的。且如果已有前导 `/`，结果会是 `//images/...`，Coil 仍会正常处理双斜杠 | 低 |
| `serverUrl` 含尾随 `/` | `trimEnd('/')` 已去除 | 低 |
| 去掉标签后空状态视觉 | 空状态时整个灰色方块无文字，但点击事件仍存在，用户可点击触发上传 | 预期行为 |
| 去掉标签后图片视觉 | 图片铺满缩略图，视觉效果更好 | 优化 |
| Coil 缓存 | 修改 URL 后原有缓存不命中，但这是正常行为，下次加载会重新拉取 | 低 |
