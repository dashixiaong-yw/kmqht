### BUG-001: PickDetailViewModel.refresh() 遗漏3个新字段

**位置**: [PickDetailViewModel.kt](file:///d:/trea项目\快麦取货通/app/src/main/java/com/kuaimai/pda/ui/pickdetail/PickDetailViewModel.kt) L272-L282

```kotlin
val orderEntity = PickOrderEntity(
    id = detail.id,
    orderNo = detail.orderNo,
    ...
    expireAt = ...,           // ← 没有 createdBy/assignedTo/visibility
)
pickOrderRepository.updateOrder(orderEntity)  // ← 覆盖写入，3个字段变为默认值
```

**影响**: 进入取货单详情页 → `refresh()` → `updateOrder` 覆盖本地数据库 → `createdBy` 变 `""`、`visibility` 变 `"private"` → 取货列表卡片显示异常。

**修复**: 增加 3 个参数：

```kotlin
val orderEntity = PickOrderEntity(
    ...
    expireAt = ...,
    createdBy = detail.createdBy,
    assignedTo = detail.assignedTo,
    visibility = detail.visibility
)
```
